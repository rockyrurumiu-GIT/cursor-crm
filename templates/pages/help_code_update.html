将本地代码变更同步到 BMS 服务器并重启服务

以下命令在本地执行。一定要先 cd /Users/rocky/Documents/cursor-crm，否则 ./ 会把当前目录里的内容同步上去。
1. 进入本地代码目录 本地 
cd /Users/rocky/Documents/cursor-crm
2. 同步代码变更至 BMS 服务器 rsync

不会覆盖服务器上的 crm_v8.db、uploads/ 与 .crm_admin_credentials.json。

rsync -avz \
  --exclude '.git' \
  --exclude '.codex' \
  --exclude '.cursor' \
  --exclude 'venv' \
  --exclude 'crm_v8.db' \
  --exclude 'crm.db' \
  --exclude 'uploads' \
  --exclude '.crm_admin_credentials.json' \
  ./ rocky@47.116.194.98:/home/rocky/BMS/
3. 服务器重启 SSH 
ssh rocky@47.116.194.98
cd /home/rocky/BMS
sudo systemctl restart bms
sudo systemctl status bms --no-pager
